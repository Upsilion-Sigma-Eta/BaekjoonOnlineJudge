# 1546번 평균

## 링크

https://www.acmicpc.net/problem/1546

## 제약 조건

| 제약 조건명 |   값    |
| :---------: | :-----: |
|  시간 제한  |  2 초   |
| 메모리 제한 | 128 MB  |
|  정답 비율  | 50.255% |

## 접근

max값을 찾아낸 다음에, 점수를 고치는 공식에 따라 새 점수들을 계산하면서 새점수들의 합을 함께 계산한다. 새 점수들의 합 계산이 완료되면 N으로 나눠서 평균을 구하고 출력하면 된다. 자료형은 전부 double로 통일하면 된다.
