# 2866번 문자열 잘라내기

## 링크

[2866번: 문자열 잘라내기 (acmicpc.net)](https://www.acmicpc.net/problem/2866)

## 제약 조건

| 제약 조건명 |   값    |
| :---------: | :-----: |
|  시간 제한  |   1초   |
| 메모리 제한 |  256MB  |
|  정답 비율  | 39.297% |

## 접근

이분탐색을 이용해서 일정 줄 수 만큼을 잘라내고, 문자열을 확인해본다. 만약 조건에 부합하는 문자열이 생성될 수 있다면, left를 1 증가시키고, 그렇지 않다면 right를 1씩 감소시킨다. 이후 탐색이 종료된 시점에서 right의 값이 잘라낼 수 있는 최대 문자열의 수이다. 잘라낸 문자열이 조건에 맞는지를 확인할 때에는 Python에서는 set를 활용하면 간편하게 구현할 수 있다.
