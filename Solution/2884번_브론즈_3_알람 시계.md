# 2884번 알람 시계

| 제약 조건명 |   값    |
| :---------: | :-----: |
|  시간 제한  |   1초   |
| 메모리 제한 |  128MB  |
|  정답 비율  | 37.741% |

## 접근

현재 분에서 45분을 뺀다. 만약 뺀 분의 수가 0 미만이라면, 시간을 한 시간 감소시키고 분에 60을 더한다. 이는 뺄셈에서 윗 자릿수의 값을 하나 빌려오는것과 동일하다. 시간의 경우에도 현재 시간이 0 미만일 경우에는 시간에 24를 더한다. 이렇게 하면 45분전의 식산을 알 수 있다.
