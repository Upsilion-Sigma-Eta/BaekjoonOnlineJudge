[3052번: 나머지 (acmicpc.net)](https://www.acmicpc.net/problem/3052)

## 제약 조건

| 제약 조건명 |   값    |
| :---------: | :-----: |
|  시간 제한  |   1초   |
| 메모리 제한 |  128MB  |

## 접근

나머지를 담을 집합을 하나 만들고, 해당 집합에 10개 수에 대해서 가각 `mod 42`를 수행한 결괏값을 넣은 다음, 집합의 원소의 갯수를 출력하면 된다.
