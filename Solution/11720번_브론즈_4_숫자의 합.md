# 11720번 숫자의 합

## 링크

[11720번: 숫자의 합 (acmicpc.net)](https://www.acmicpc.net/problem/11720)

## 제약 조건

| 제약 조건명 |   값    |
| :---------: | :-----: |
|  시간 제한  |   1초   |
| 메모리 제한 |  256MB  |
|  정답 비율  | 56.005% |

## 접근

입력받은 숫자 N개를 문자열로 취급한다. 이 문자열을 S라고 할 때, 문자열 S의 각 문자를 int로 변환하면서 더한 다음에, 결괏값을 출력하면 된다.
