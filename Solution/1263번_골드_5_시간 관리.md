# 1263번 시간 관리

## 링크

[1263번: 시간 관리 (acmicpc.net)](https://www.acmicpc.net/problem/1263)

## 제약 조건

| 제약 조건명 |   값    |
| :---------: | :-----: |
|  시간 제한  |   2초   |
| 메모리 제한 |  128MB  |
|  정답 비율  | 43.428% |

## 접근

작업들을 마감시간이 늦은 순서대로 정렬하고, 현재 시간을 가능한 최대 마감시간에서부터 시작해서 각 처리시간을 빼가면서 시간을 앞으로 당기면 된다. 시각적으로 생각해보자면, 막대를 짧은 순서대로 하나의 줄의 가장 오른쪽에 채워넣고, 한 줄의 가장 오른쪽에서 시작해서 각 막대들의 처리시간만큼 앞으로 이동하는 것이라고 생각해볼 수 있다. 이 때, 전체를 처리하기가 어렵다면 현재 시간은 가장 왼쪽을 넘어갈 것이고, 전체를 시간 안에 처리할 수 있다면 줄 막대의 가장 왼쪽이 가리키는 지점이 가장 늦게 시작할 수 있는 시간이 될 것이다.
