# 1152번 단어의 개수

## 링크

[1152번: 단어의 개수 (acmicpc.net)](https://www.acmicpc.net/problem/1152)

## 제약 조건

| 제약 조건명 |   값    |
| :---------: | :-----: |
|  시간 제한  |   2초   |
| 메모리 제한 |  128MB  |
|  정답 비율  | 33.186% |

## 접근

중복된 단어가 등장하더라도 단어의 개수가 올라간다는 점에 유의한다. 문자열을 받아서 맨 좌측과 우측의 공백을 잘라내는 처리를 거치고, 리스트를 공백을 기준으로 분할한 뒤, 리스트의 원소의 개수를 출력하면 문제를 해결할 수 있다.
